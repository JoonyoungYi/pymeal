# pymeal

## 개요
pymeal은 나이스 학생서비스에 접속해 급식을 가져오는 라이브러리입니다.  
나이스에 등록되어 있지 않은 급식은 표시되지 않을 수 있습니다.

## 설치
**Python 3.0 이상에서만 동작합니다**  
`pip install pymeal`으로 간단하게 설치하실 수 있습니다
